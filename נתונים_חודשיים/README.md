# נתונים חודשיים — i24

תיקייה לקבצי הדאטה החדשים שמגיעים מ-i24 פעם בחודש.

## איך משתמשים

1. שמרי את הקובץ החודשי כאן (csv או xlsx)
2. תני שם תיאורי — לדוגמה: `i24_מאי_2026.csv` או `broadcasts_2026-05.xlsx`
3. אמרי לקלוד: "יש לי קובץ חדש בנתונים_חודשיים"
4. קלוד יריץ את הסקיל `process-i24-data` שיעשה את כל השאר

## מה הקובץ צריך להכיל

15 עמודות (אותו פורמט כמו `רשימת תוכניות.csv` המקורי):

```
שם תוכנית · יום שידור · תאריך שידור · שעת התחלה · שעת סיום ·
משך תוכנית · רייטינג · נתח · צופים 4+ · חשיפה 4+ · משך צפייה ·
כאן 11 · קשת 12 · רשת 13 · עכשיו 14
```

CSV עם שורת באנר עליונה — נתמך גם כן.

## דוגמה לקובץ ייחוס

`../רשימת תוכניות.csv` — המבנה המדויק שמצופה.
